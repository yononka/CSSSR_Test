# CSSSR_Test

Для запуска тестов необходима среда разработки PyCharm https://www.jetbrains.com/pycharm/download/#section=windows
Нужно установить версию Community

При первом открытии PyCharm выбрать Check out from Version Control > Git > вставить в поле url проекта https://github.com/yononka/CSSSR_Test

Открыть файл test_Superheroes.py и запустить его с помощью pytest, который должен быть доступен и интегрирован вместе с проектом.

К сожалению, из-за ошибки в API, которая проявляется при повторном обращении к ендпоинтам по адресу https://superhero.qa-test.csssr.com/" объективность прохождения тестов нарушена.
